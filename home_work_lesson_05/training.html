CSS block properties

padding

Властивість padding — це внутрішній відступ. Використовується для відступів від кордону блоку всередину блоку (елементу), а не між блоками (елементами). Значення можуть бути лише додатними, тобто більше або рівне нулю.
Простими словами padding — це Відступи ВСЕРЕДИНІ об'єкту, від кордонів об'єкта ДО ЙОГО КОНТЕНТУ. НЕ успадковується.
При вказанні значення у відсотках, відсоток береться від ширини батьківського об'єкта, навіть для вертикальних відступів.

margin

Властивість margin — Зовнішній відступ. Використовується для відступів між блоками (елементами), рекомендується робити відступи зверху вниз (margin-bottom) зліва на право. Значення можуть бути від'ємними! НЕ успадковується.
Правило для відступів margin між елементами — відступи margin задаємо на одному рівні вкладеності.
Не допускається відступ від дочірнього елементу до наступного батьківського, тобто відступ з 2-го рівня вкладеності до першого.

width

Властивість width — це ширина блоку (елементу). Значення можуть бути як у відсотках, так і у пікселях. При вказанні значення у відсотках, відсоток береться від ширини батьківського об'єкта. НЕ успадковується.НЕ працює з рядковими об'єктами.
Властивість `max-width` – встановлює **максимальну ширину** блочних (блочно-рядкових) тегів, тобто блок не буде більшим за вказану ширину. Не успадковується. НЕ працює зі рядковими об'єктами.
Властивість `min-width` — встановлює **мінімальну ширину** блочних (блочно-рядкових) тегів, тобто блок не буде меншим за вказану ширину. Не успадковується. НЕ працює зі рядковими об'єктами.

height

Властивість `height` — встановлює чітку висоту блочних (блочно-рядкових) тегів. Не успадковується. НЕ працює зі рядковими об'єктами.
Властивість `min-height` — встановлює мінімальну висоту блочних (блочно-рядкових) тегів, тобто блок не буде меншим за вказану висоту. Не успадковується. НЕ працює зі рядковими об'єктами.
Властивість `max-height` — встановлює максимальну висоту блочних (блочно-рядкових) тегів, тобто блок не буде більшим за вказану висоту. Не успадковується. НЕ працює зі рядковими об'єктами.

box-sizing

Застосовується для зміни алгоритму обчислення величин. Значення властивості: - content-box - (за замовченням) додає відступи до розміру об'єкта; - border-box - значення розмірів враховують відступи.
ВАЖЛИВО! Видалення фантомних відступів між елементами. Для батьківського елементу вказуємо font-size: 0; це прибере фантомний відступ між елементами яким вказано display: inline-block; а для дочірніх елементів потрібно вказати відповідний розмір шрифту, приклад font-size: 16px;

display

Властивість `display`— багатоцільова властивість яка задає глобальні параметри тегу, не успадковується. ВАЖЛИВО! Значення властивості: - block — елемент відображається як блочний (встановлено за замовченням для блочних об'єктів); - inline— елемент відображається як рядковий (встановлено за замовченням для рядкових об'єктів); - inline-block — елемент відображається як рядковий, але з можливістю використання усіх CSS-властивостей (блочно-рядковий); - none — елемент не виводиться в браузер. Інші елементи займають його місце. Не бере участь в обробці браузером.

overflow

Властивість `overflow` — керує відображенням контенту блочного елементу. Не успадковується. ВАЖЛИВО! Значення властивості: - visible — відображається увесь контент елемента, навіть за межами встановлених розмірів (за замовченням); - hidden — відображається тільки контент всередині (блоку) елемента, все інше буде сховано; - scroll — завжди додаються смужки прокрутки; - auto — смужки прокрутки додаються лише за необхідністю (якщо контент не вміщується всередині вказаних розмірів блоку).

vertical-align

Властивість `vertical-align` — керує вирівнюванням блочно-рядкових та рядкових елементів по вертикалі (осі - Y).
Не успадковується. - top — вирівнює по верхньому краю - middle — вирівнює по центру - bottom — вирівнює по нижньому краю - також можна задати значення у пікселях.
