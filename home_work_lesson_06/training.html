Властивість outline — встановлює товщину, стиль та колір контуру елемента. Властивість outline — встановлює товщину, стиль та колір обвідки навколо елемента (не впливає на оточення). Властивість box-shadow — встановлює тінь елемента. Не успадковується. Вона може бути як і зовні об’єкту, так і всередині

Opacity

Властивість opacity — встановлює прозорість елемента. Не успадковується. Значення від 0 до 1

0 — повна прозорість

1 — повна непрозорість (за замовченням)

0.5 — прозорість 50%.

Visibility

Властивість visibility — встановлює видимість елемента. Не успадковується. Елемент з властивістю visibility: hidden; не буде видно, але він займатиме місце на сторінці.

background

Дозволяє одночасно задати кілька параметрів фону:

  • визначити колір;
  • встановити фонове зображення;
  • задати початкове положення фонового зображення;
  • визначити як повторюватиметься фонове зображення;
  • встановити, чи прокручуватиметься фонове зображення разом із вмістом;
  • масштабувати фонове зображення відповідно до заданих розмірів;
  • вказати градієнтну заливку фону.

Властивість background-color встановлює колір фону елемента

Властивість background-image встановлює фонове зображення елемента
background-image: url("../img/ .jpg");
background-image: linear-gradient(55.95deg, #FF3F3A 0%, #F75E05 100%); 

Властивість background-position встановлює початкове положення фонового зображення. background-position: 0 0; перше значення — розташування по горизонталі (значення може бути у відсотках, px, словом), друге значення — по вертикалі. Значення:
- 0 0 (або left top) — за замовченням (зображення буде починатись з верхнього лівого кута блоку); - center (або 50%) — центрує зображення посередині блоку; - top; - left; - bottom; - right. Якщо задано значення лише для горизонталі чи лише по вертикалі то друге значення за замочуванням буде center

Властивість background-repeat визначає, як повторюватиметься фонове зображення. Значення:
- repeat — за замовчуванням (зображення буде повторюватись по горизонталі та вертикалі); - no-repeat — зображення не буде повторюватись; - repeat-x — зображення буде повторюватись лише по горизонталі; - repeat-y — зображення буде повторюватись лише по вертикалі; - space — зображення буде повторюватись, але між ними не буде проміжків, а порожні ділянки будуть розподілені рівномірно; - round — зображення буде повторюватись, але розмір зображень буде змінено так, щоб вони повністю заповнили блок без обрізання та проміжків.

Властивість background-attachment встановлює, чи прокручуватиметься фонове зображення разом із вмістом. Значення:
- scroll — за замовчуванням (фонове зображення прокручуватиметься разом із вмістом); - fixed — фонове зображення залишатиметься на місці, а вміст буде прокручуватись поверх нього; - local — фонове зображення прокручуватиметься разом із вмістом елемента, якщо у нього є власна прокрутка

Властивість background-size встановлює розмір фонового зображення. Значення:
- auto — за замовчуванням (оригінальний розмір зображення); - cover — масштабування зображення так, щоб воно повністю покривало блок (може обрізатись); - contain — масштабування зображення так, щоб воно повністю помістилось у блок (можуть залишитись порожні ділянки); - width height — можна задати конкретні розміри у px, %, em тощо. Якщо вказано лише одне значення, друге буде auto.

Універсальна властивість `background` (універсальний запис — скорочений запис властивості) в більшості випадків використовується для додавання фонового зображення, позиціювання його, розмір зображення та чи повторюється зображення. Приклад запису ⬇️
`background: green url("#") 0 0 / cover no-repeat scroll;`
- `green` - фоновий колір
- url("#")`- посилання на картинку
- `0 0`- позиція картинки відносно блоку (числові чи словесні значення)
- `/` - роздільник, він потрібен якщо вказується розмір для фонового зображення, без нього буде помилка в записі
- `cover` - розмір для картинки (окрім `cover` та `contain` можуть бути інші значення у відсотках, пікселях чи інших одиницях виміру)
- `no-repeat` - вказує чи повторюється зображення
- `scroll` - (**за замовчуванням**) вказує чи буде скролитись картинка разом з блоком
